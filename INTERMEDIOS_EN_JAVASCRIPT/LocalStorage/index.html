💾 LocalStorage Sencillo

🔍 ¿Qué es LocalStorage?

LocalStorage es como una libreta de notas digital que:

  • 📝 Guarda información en tu navegador
  • 🔄 NO se borra cuando cierras la página
  • 💾 Persiste hasta que tú la borres
  • 🔒 Es privada de cada sitio web
