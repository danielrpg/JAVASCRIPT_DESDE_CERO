El método document.write()

Historia Interactiva

Este juego demuestra el uso del método document.write() creando una historia interactiva donde cada elección genera una nueva página.

¿Qué es document.write()? Es un método que permite escribir contenido HTML directamente en el documento. Cuando se usa después de que la página ha cargado, reemplaza todo el contenido anterior.

IMPORTANTE: Al hacer una elección, todo el contenido actual será reemplazado con una nueva "página" de la historia.

document.write('<h1>Nuevo contenido</h1>');
document.write('<p>Este texto reemplazará todo lo anterior</p>');
