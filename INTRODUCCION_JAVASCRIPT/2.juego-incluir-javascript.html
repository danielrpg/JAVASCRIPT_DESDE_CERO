¿Cómo y dónde incluir JavaScript?

Este juego muestra las diferentes formas de incluir JavaScript en una página HTML.

Método 1: En la cabecera (head)

<head>
  <script>
    function metodo1() {
      document.getElementById("resultado1").innerHTML = "...";
    }
  </script>
</head>

Método 2: En el cuerpo (body)

<body>
  <script>
    function metodo2() {
      document.getElementById("resultado2").innerHTML = "...";
    }
  </script>
</body>

Método 3: JavaScript en línea (inline)

<button onclick="document.getElementById('resultado3').innerHTML = '¡Funcionó!...';">

Método 4: Desde archivo externo

<script src="script.js"></script>

Nota: En este ejemplo, simulamos un archivo externo.
