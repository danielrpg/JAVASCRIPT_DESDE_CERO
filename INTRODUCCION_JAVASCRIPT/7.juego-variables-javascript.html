Variables utilizadas en este ejemplo:

Este juego utiliza diferentes tipos de variables para gestionar la tienda:

  • let carrito = []; - Array para almacenar los productos en el carrito
  • let total = 0; - Variable numérica para el total de la compra
  • const IVA = 0.15; - Constante para el porcentaje de impuesto
  • const DESCUENTO = 0.10; - Constante para el porcentaje de descuento
  • let descuentoAplicado = false; - Variable booleana para controlar si ya se aplicó el descuento
  • let impuestoAplicado = false; - Variable booleana para controlar si ya se aplicó el impuesto
